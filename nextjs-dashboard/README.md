# ~ CAMPUS COMPANION WEB APP ~

## Overview
This project is a web-based "Campus Companion" app designed to support students and staff in daily campus life. It includes these features: helpdesk ticket system, canteen menu, society events, timetable, and accessibility settings.

## Features
- Helpdesk ticket system (submit and manage issues)
- Event and timetable pages (in progress)
- Canteen Menu
- Society Events
- Accessibile design

## My Contribution (Peace)

## Technologies Used

- TypeScript
- Tailwinf CSS

## How to run the project: 
1. CLone repo: git clone <repo-link>
2. Navigate into project: cd nextjs-dashboard
3. Install dependencies: pnpm install
4. Run development server: pnpm dev
5. Open the link in browser: localhost
